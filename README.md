# Manish Neupane - Portfolio

Portfolio website showcasing projects in physics, computer science, and robotics.

Built with React, Material-UI, and deployed on Cloudflare Pages.

## Tech Stack

- React 19
- Material-UI (MUI)
- React Router
- Framer Motion
- Deployed on Cloudflare Pages

## Local Development

```bash
npm install
npm start
```

Visit `http://localhost:3000`

## Build

```bash
npm run build
```
